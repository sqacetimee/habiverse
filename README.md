# Habiverse

Explore 5,000+ confirmed exoplanets using real NASA data.

**[Live →](https://exoplanet-explorer-teal.vercel.app)**

Built with Next.js 15, TypeScript, and the NASA Exoplanet Archive TAP API. Features a 6-factor habitability scoring algorithm with an Atmosphere Retention Metric (ARM) based on Zahnle & Catling (2017).
